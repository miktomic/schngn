# Visionary sketch comparison

Generated four disposable money-shot variants for SCHNGN.

## Design skills / references used

- `sketch`: standalone HTML variants + README per direction.
- `claude-design`: surface-first composition and anti-slop audit discipline.
- `popular-web-designs`: concrete visual vocabularies from Apple, IBM/Carbon, Linear, Framer, and Notion.
- Web research:
  - Jony Ive: care, obviousness, object-like simplicity.
  - Edward Tufte: high data-ink, direct labeling, truthfulness, complexity with clarity.
  - Bret Victor: explorable explanations, direct manipulation, active reading.
  - Dieter Rams: useful, understandable, honest, unobtrusive, less but better.

## Variants

| Variant | Surface | Core idea | Main interaction | Likely role |
|---|---|---|---|---|
| `002-ive-calm-permission-object` | Decide / Permission | One premium answer card | Flip card + stress test | Consumer home |
| `003-tufte-evidence-ledger` | Inspect | Answer and evidence visible together | Scenario toggle + row inspection | Explanation / report |
| `004-victor-explorable-simulator` | Explore / Manipulate | Drag the plan, see the rule react | Slider changes Italy length live | Flagship planner |
| `005-rams-compliance-appliance` | Operate / Instrument | Compliance readout with obvious controls | Run +9 day test | Serious/current status |

## First recommendation

Use a hybrid:

1. **Ive** for the default “am I safe?” home surface.
2. **Victor** for the planning interaction.
3. **Tufte** for explanation and border-ready report.
4. **Rams** for current-status/admin mode and professional credibility.

The likely consumer winner is **Ive + Victor**. The likely paid/pro winner is **Tufte + Rams**.

# SCHNGN synthesis from UI UX Pro Max

## What the skill recommended

The strongest useful signals from UI UX Pro Max for SCHNGN:

- Style: **Trust & Authority**, with Swiss/Minimal dashboard influence.
- Palette: high-contrast navy/blue with semantic green/red status colors.
- Typography: public-service readability: Lexend / Source Sans 3 is stronger for SCHNGN than Fira Code/Fira Sans.
- Charting: prefer **bullet chart / progress-to-target** for 75/90 vs 90, not a decorative circular gauge.
- UX priorities: no horizontal scroll, 44px+ touch targets, reduced-motion support, no hover-only states, accessible contrast.

## What I reject

The generic design-system output suggested an “App Store Style Landing” for some queries. That is not the money-shot app UI. Useful for a marketing page later, not for the core planner screen.

The initial operations-dashboard recommendation was directionally useful but too dark/SOC-like for a traveler. SCHNGN needs trust without looking like airport cybersecurity software having a difficult childhood.

## Variant: Rams Compliance Appliance

### Design stance
A serious instrument panel for avoiding legal/admin trouble: useful, honest, understandable.

### Key choices
- Layout: status panel, readout cells, trip table, direct controls.
- Typography: IBM Plex / Carbon-like precision.
- Color: monochrome with semantic blue, green, red.
- Interaction: run +9 day test toggles permitted/fault condition.

### Trade-offs
- Strong at: credibility, clarity, older/professional users.
- Weak at: less emotionally warm; may feel bureaucratic.

### Best for
- Current-status mode, advisor mode, serious reporting.

## Variant: Victor Explorable Simulator

### Design stance
An explorable explanation: users manipulate the trip and immediately see the Schengen rule react.

### Key choices
- Layout: timeline workspace first, result card follows the manipulation.
- Typography: modern dark product/tool aesthetic.
- Color: dark Framer/Linear-inspired canvas with green safe and red violation.
- Interaction: range slider changes Italy length and recalculates visible state.

### Trade-offs
- Strong at: differentiation, teaching the rolling-window rule, planning.
- Weak at: touch interaction complexity; must be extremely polished to avoid frustration.

### Best for
- Flagship trip planner interaction.

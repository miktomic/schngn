## Variant: Ive Calm Permission Object

### Design stance
A premium single-object decision screen: answer first, reveal machinery only on request.

### Key choices
- Layout: one dominant card with a subtle timeline embedded inside it.
- Typography: large SF-style system type and tight hierarchy.
- Color: neutral Apple-like surfaces with restrained green/amber status.
- Interaction: card flips to reveal the explanation; stress test toggles safe/risk.

### Trade-offs
- Strong at: reducing anxiety, feeling polished, fast consumer comprehension.
- Weak at: hiding calculation detail; may need stronger trust affordances.

### Best for
- Default mobile home screen for consumer users.

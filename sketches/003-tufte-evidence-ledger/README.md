## Variant: Tufte Evidence Ledger

### Design stance
A report-like evidence surface where the answer and proof are visible together.

### Key choices
- Layout: annotated ledger + compact timeline.
- Typography: paper/report feel with sparse sans labels.
- Color: monochrome, red only for violation, purple only for what-if.
- Interaction: scenario toggle and inspectable row labels.

### Trade-offs
- Strong at: trust, audits, border-ready report logic.
- Weak at: emotional calm; may feel dense for casual mobile users.

### Best for
- Explanation mode, PDF report, advisor/pro workflows.
